# 윈바신갈지역아동센터 모바일 웹앱

## 실행 방법
1. `index.html`을 크롬에서 열면 화면을 확인할 수 있습니다.
2. 휴대폰 앱처럼 설치하려면 Netlify, Vercel, GitHub Pages 같은 HTTPS 서버에 폴더 전체를 업로드합니다.
3. 안드로이드 크롬에서 접속한 뒤 `홈 화면에 추가` 또는 화면 하단의 `설치` 버튼을 누릅니다.

## 버튼 연결
- 홈페이지: https://www.hjy.kr/center/index.php?cId=em-pk
- 좋은가게: https://www.coupang.com/

## 주의
쿠팡 파트너스 문구는 화면 하단에 넣어 두었습니다. 실제 파트너스 링크를 사용하려면 쿠팡 파트너스에서 발급받은 전용 링크로 변경해야 합니다.
